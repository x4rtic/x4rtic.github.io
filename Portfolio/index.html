Bushido: Fallen Era

Bushido: Fallen Era is an independent MMORPG set in a dark, feudal-inspired world. The project emphasizes player agency, strategic real-time combat, and long-term progression in a persistent online environment.

  • Classless progression system
  • Persistent open world with dynamic events
  • Skill-based real-time combat
  • Player-driven factions and territory control
  • Handcrafted pixel art and custom engine systems

Development is in progress. Core systems such as networking, world simulation, and combat logic are being built from the ground up.
